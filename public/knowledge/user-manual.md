# IslaPOS User Manual

## English

### 1) Login & Setup
**Goal:** Create your account and finish initial setup.

1. Go to **Sign In** and choose **Create account**.
2. Verify your email.
3. Complete **Setup**:
   - Restaurant name
   - Location and timezone
   - IVU (Puerto Rico tax)
4. You can return to Setup anytime from **Admin**.

### 2) Menu Setup
**Goal:** Build your categories, products, and modifiers.

1. Go to **Admin → Menu**.
2. Create categories (e.g., Appetizers, Mains, Drinks).
3. Add products with price, description, SKU/barcode (optional).
4. Add modifiers in **Admin → Modifiers** (e.g., Size, Extras).
5. Assign modifiers to products.

### 3) POS Orders
**Goal:** Create and close orders quickly.

1. Go to **POS**.
2. Add items by category or Search.
3. Choose order type: **Counter / Pickup / Delivery / Dine‑in**.
4. Review subtotal, tax, and total.
5. Select payment method and **mark as paid**.

### 4) Tables & Floor Plan
**Goal:** Manage table service.

1. Go to **POS → Tables** to open tickets by table.
2. For visual layout, go to **Admin → Floor Plan**.
3. Create areas (Main Salon, Patio, Bar).
4. Drag tables, set **seats, size, and shape**.

### 5) KDS (Kitchen Display)
**Goal:** Show orders to the kitchen.

1. Go to **Admin → KDS QR Codes**.
2. Create a link per station (Kitchen, Bar).
3. Print or show the QR code.
4. Kitchen staff scans QR to open KDS.

## Español

### 1) Login y Configuración
**Objetivo:** Crear cuenta y completar configuración inicial.

1. Ve a **Iniciar sesión** y elige **Crear cuenta**.
2. Verifica tu correo.
3. Completa **Setup**:
   - Nombre del restaurante
   - Ubicación y zona horaria
   - IVU (impuesto de Puerto Rico)
4. Puedes volver a Setup desde **Admin** cuando quieras.

### 2) Configuración del Menú
**Objetivo:** Crear categorías, productos y modificadores.

1. Ve a **Admin → Menú**.
2. Crea categorías (ej. Entradas, Platos, Bebidas).
3. Agrega productos con precio, descripción, SKU/código (opcional).
4. Agrega modificadores en **Admin → Modificadores** (ej. Tamaño, Extras).
5. Asigna modificadores a productos.

### 3) Órdenes en POS
**Objetivo:** Crear y cerrar órdenes rápido.

1. Ve a **POS**.
2. Agrega productos por categoría o búsqueda.
3. Elige el tipo: **Mostrador / Pickup / Delivery / Mesa**.
4. Revisa subtotal, IVU y total.
5. Selecciona método de pago y **marca como pagada**.

### 4) Mesas y Plano de Piso
**Objetivo:** Manejar servicio de mesa.

1. Ve a **POS → Mesas** para abrir tickets.
2. Para un layout visual, ve a **Admin → Floor Plan**.
3. Crea áreas (Salón, Patio, Barra).
4. Arrastra mesas y configura **sillas, tamaño y forma**.

### 5) KDS (Pantalla de Cocina)
**Objetivo:** Mostrar órdenes en cocina.

1. Ve a **Admin → KDS QR Codes**.
2. Crea un link por estación (Cocina, Barra).
3. Imprime o muestra el QR.
4. El personal escanea el QR para abrir KDS.
